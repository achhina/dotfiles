# Timeline

## When to Use

Use timelines for:
- Chronological event sequences
- Product roadmap visualization
- Historical milestones
- Release history

## Example

```mermaid
%%{init: {'theme':'forest'}}%%
timeline
    title Product Evolution Timeline
    section 2023
        Q1 : MVP Launch
           : 1000 Users
        Q2 : Mobile App Release
           : API v2
           : 10K Users
        Q3 : Enterprise Features
           : SSO Integration
        Q4 : Analytics Dashboard
           : 50K Users
    section 2024
        Q1 : AI-Powered Search
           : Multi-language Support
        Q2 : Marketplace Launch
           : 100K Users
        Q3 : Advanced Reporting
           : Custom Integrations
        Q4 : Platform API
           : Developer Portal
```

## Key Conventions

- Use `section` to group time periods (years, quarters, months)
- Each entry represents an event or milestone
- Use `:` to separate multiple events in the same period
- Keep event descriptions concise
- Order sections chronologically
- Include metrics or achievements when relevant
- Use consistent time period granularity
